How to ROOT and UNROOT Xiaomi Redmi 1S MIUI ROM

Prequisites:

1)The Redmi 1S must be running the Stock MIUI ROM.

2) Developer options should be activated. If not, then go to Settings > About Phone > and tap the build number five times to enable developer option).

3) It’s strongly recommended to create the manual back up of all the data present in your Redmi.
